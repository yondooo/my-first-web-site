HTML

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and wep wep applications.

Web browsers receive HTML documnets from a web server or from local strogate and render them into multimedia web pages.


HTML elements are the building blocs of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page.
